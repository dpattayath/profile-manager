
# Profile Manager

Web app for managin talent profiles

- Lists profiles
- Filter based on location and category
- Delete profile once logged in

## Tech Stack
- api
    - PHP on Laravel 7
- ui
    - React 16.13

## How To

Once cloned run from the root directory
```
docker-compose up -d
```

and navigate on browser:
```
http://localhost:3000
```
## Authentication

Static single admin account set up
```
admin@email.com
admin123
```
